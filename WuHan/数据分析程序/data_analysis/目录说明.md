# 当前项目结构

## 入口
- `main.py`：主入口，按顺序调度清洗、分析、词云、地图和聚类
- `analysis_common.py`：共享配置、数据加载和标准化导出

## 数据流
- `preprocessing/test_clean.py`：标准化原始武汉挂牌数据
- `artifacts/ershoufang-wuhan-standard.csv`：标准化后的主输入数据
- `analysis/baidu_map.py`：生成 `artifacts/latlng.csv` 和 `artifacts/star.txt`
- `clustering/run.py`：读取 `artifacts/latlng.csv` 做聚类并输出结果

## 分析脚本
- `analysis/price_and_area.py`：价格、面积、区域统计
- `analysis/house_attr.py`：户型、朝向、面积分布
- `analysis/business_attr.py`：用途分布
- `analysis/test_ana.py`：综合演示图表
- `analysis/ciyun.py`：词云

## 资源
- `assets/`：背景图、字体等静态资源
- `analysis/picture/`：词云等图片输出
- `clustering/result/map/`：聚类输出的地图文件

## 关系说明
1. `main.py` 先调用 `preprocessing/test_clean.py` 生成标准化数据。
2. 各分析脚本读取 `artifacts/ershoufang-wuhan-standard.csv`。
3. 地图脚本在有 `BAIDU_MAP_AK` 时生成坐标文件。
4. 聚类脚本依赖坐标文件，输出聚类结果。
